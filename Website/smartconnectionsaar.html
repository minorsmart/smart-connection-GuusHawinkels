Smart connection

Situation: HTML en CSS waren nieuw voor mij. Mijn opleiding Mechatronica ook gericht is op programmeren, maar niet op HTML en CSS.

Task: Het leren van HTML en CSS en het gebruik ervan om een portfolio website te maken.

Action: Ondanks dat ik al redelijk wat programmeerkennis had van mijn opleiding in mechatronica, heb ik actief deelgenomen aan het leerproces van HTML en CSS. Ik heb bronnen en tutorials geraadpleegd om de basisprincipes van HTML en CSS te begrijpen en ben begonnen met het bouwen van mijn portfolio website. Ik heb nieuwe concepten en technieken verkend en toegepast om mijn website te ontwerpen en vorm te geven.

Result: Als resultaat van mijn inzet heb ik HTML en CSS geleerd en met succes een portfolio website gemaakt. Ik ben nu bekend met de basisprincipes en mogelijkheden van HTML en CSS, wat mij in staat stelt om webpagina's te ontwikkelen en vorm te geven. Het was een interessante ervaring om met deze programmeertalen te werken en het heeft mijn vaardigheden op het gebied van webontwikkeling vergroot.
